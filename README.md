# FOSEP Himalayas

Website for the **Federation of Societies for Environmental Protection (FOSEP)** —
an environmental and community NGO in the Darjeeling Hills & Eastern Himalayas.

A premium revamp of [fosepdarjeeling.org](https://fosepdarjeeling.org/), built as a
lightweight vanilla HTML/CSS/JS site hosted on GitHub Pages.

## Develop locally

No build step. Serve the folder with any static server, e.g.:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```
